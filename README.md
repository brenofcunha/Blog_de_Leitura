# Blog_de_Leitura

Projeto para desenvolver um blog pessoal das leituras realizadas, em Django, com foco visual.

## Como rodar localmente

```bash
python -m pip install -r requirements.txt
python manage.py runserver
```

## Páginas iniciais

- `/` — página de entrada estilizada.
- `/programador-pragmatico/` — espaço inicial para os resumos do livro *Programador Pragmático*.
